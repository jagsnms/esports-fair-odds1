# Banked initiatives (deferred)

This file is the minimal registry of **reviewed, deferred** deep initiatives so the planner does not repeatedly re-propose them without new evidence.

## Replay + Simulation Validation Architecture

- **status**: banked
- **decision**: reviewed and deferred
- **reason**: valuable future initiative, but not the current next implementation priority
- **reopen_when**:
  - replay-validation stages are exhausted, or
  - new evidence makes simulation the immediate blocker, or
  - calibration/live validation no longer outrank it
